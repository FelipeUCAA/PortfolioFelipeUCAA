Projeto M.A.R.I.O

Desenvolvi um jogo inspirado no universo do Mario para demonstrar minhas habilidades em programação front-end. Neste jogo de plataforma simples, os jogadores controlam o Mario e devem evitar obstáculos, representados por canos, enquanto tentam acumular a maior pontuação possível. O jogo requer timing e reflexos para evitar colisões, proporcionando uma experiência divertida e desafiadora. Além disso, adicionei elementos de interface gráfica e animações para tornar a experiência mais imersiva e cativante.

Projeto Mapa Web Interativo

Um mapa Web com o intuito de facilitar a visualização de dados do SIGEF e suas devidas áreas no mapa do Brasil, com integração de busca de locais/coordenadas, geolocalização, busca automática de coordenadas e botões de camadas, além de uma atualização de dados automática, usando o "blob storage" da plataforma Azure. Um projeto extenso que visa ajudar os usuários a visualizar o Sistema de Gestão Fundiária brasileiro de forma prática.

Bot de Liquidez Uniswap

Este projeto implementa um bot de liquidez para a Uniswap, capaz de implantar um contrato inteligente ERC-20 em redes de teste como Sepolia e interagir com a DEX para fornecer liquidez e realizar trocas de tokens. Desenvolvido com Truffle e Node.js, o bot automatiza operações essenciais de DeFi, integrando serviços como Infura, Alchemy e Metamask para facilitar a conexão com a blockchain Ethereum.
