M.A.R.I.O Project

I developed a game inspired by the Mario universe to demonstrate my front-end programming skills. In this simple platform game, players control Mario and must avoid obstacles, represented by pipes, while trying to accumulate the highest possible score. The game requires timing and reflexes to avoid collisions, providing a fun and challenging experience. I also added graphical interface elements and animations to make the experience more immersive and captivating.

Interactive Web Map Project

A Web map designed to facilitate the visualization of SIGEF data and its respective areas on the map of Brazil, with integration for searching locations/coordinates, geolocation, automatic coordinate search, and layer buttons, in addition to automatic data updating using Azure's "blob storage" platform. An extensive project aimed at helping users practically view the Brazilian Land Management System.

Uniswap Liquidity Bot

This project implements a liquidity bot for Uniswap, capable of deploying an ERC-20 smart contract on test networks like Sepolia and interacting with the DEX to provide liquidity and perform token swaps. Developed with Truffle and Node.js, the bot automates essential DeFi operations, integrating services like Infura, Alchemy, and Metamask to facilitate connection with the Ethereum blockchain.
